# code.charliegleason.com

A personal code blog built with Astro 6 and deployed on Cloudflare Workers.

## Tech Stack

- **Framework**: Astro 6 with SSR
- **Runtime**: Cloudflare Workers
- **Content**: MDX with React components
- **Styling**: Tailwind CSS v4
- **Code Highlighting**: Shiki
- **Interactive Code**: Sandpack
- **AI Visuals**: Replicate API

## Development

```bash
npm install    # Install dependencies
npm run dev    # Start dev server (localhost:4321)
```

## Commands

| Command | Action |
| :------ | :----- |
| `npm run dev` | Start dev server |
| `npm run build` | Build for production |
| `npm run preview` | Preview production build |
| `npm run typecheck` | Run type checking |
| `npm run lint` | Check code style |
| `npm test` | Run tests |

## Visual Generation

```bash
npm run generate:images   # Generate AI images
npm run generate:videos   # Generate AI videos  
npm run generate:colors   # Extract colors from images
```

## Deployment

```bash
npm run deploy           # Deploy to production
npm run deploy:staging   # Deploy to staging
```
